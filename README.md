# AI Test Generator

Modern educational test generator powered by Google Gemini AI.

## Features

- **AI-Powered**: Generates questions based on provided source text using Gemini AI.
- **Multiple Question Types**:
  - Single Choice
  - Multiple Choice
  - Matching
  - Sorting
- **DOCX Export**: Automatically generates properly formatted Word documents.
- **Two Variants**: Creates two distinct variants of the test with answer keys.
- **Context Aware**: Supports Grade, Lesson Number, and Source Text context.

## Usage

1. Open `index.html` in a web browser.
2. Fill in the class, lesson, topic, and source text.
3. Select the number of questions for each type.
4. Click "Generate" to download the test.
